# 우주 디펜스 게임 파워업 시스템

## 개요

우주 디펜스 게임에 추가된 파워업 시스템은 게임 플레이에 다양성과 전략적 요소를 부여합니다. 플레이어는 단어를 타이핑하여 파워업 아이템을 획득하고, 이를 통해 게임 플레이를 유리하게 이끌 수 있습니다.

## 아이템 종류

다음과 같은 다양한 파워업 아이템이 게임 내에 등장합니다:

### 1. 시간 감속 (파란색)
- **효과**: 모든 적의 속도를 15초 동안 50% 감소시킵니다.
- **아이콘**: 모래시계 (hourglass_bottom)
- **희귀도**: 일반 (70% 확률로 등장)

### 2. 다중 공격 (보라색)
- **효과**: 10초 동안 1.5초마다 자동으로 무작위 적을 공격합니다.
- **아이콘**: 별 (auto_awesome)
- **희귀도**: 중간 (50% 확률로 등장)

### 3. 보호막 (파란색)
- **효과**: 20초 동안 또는 적이 우주선에 도달할 때까지 피해를 방어합니다.
- **아이콘**: 방패 (shield)
- **희귀도**: 중간 (40% 확률로 등장)

### 4. 생명력 회복 (빨간색)
- **효과**: 생명력을 1 회복합니다 (최대 3까지).
- **아이콘**: 하트 (favorite)
- **희귀도**: 희귀 (30% 확률로 등장)

### 5. 대폭발 (주황색)
- **효과**: 화면 내 모든 적을 즉시 파괴합니다.
- **아이콘**: 불 (local_fire_department)
- **희귀도**: 중간 (35% 확률로 등장)

## 구현 세부사항

### 아이템 생성 주기
- 기본적으로 15-20초 간격으로 랜덤하게 아이템 생성
- 웨이브가 높아질수록 생성 간격이 짧아짐 (웨이브당 0.5초씩 감소)
- 아이템은 일정 확률로만 생성됨 (아이템 종류에 따라 확률이 다름)

### 아이템 표시
- 아이템은 적보다 느리게 이동하며, 적과 유사하게 단어로 표시됨
- 각 아이템은 고유한 색상과 아이콘으로 구분됨
- 아이템은 일반적으로 짧은 단어를 사용하여 빠르게 획득 가능

### 효과 관리
- 활성화된 효과는 상단 바에 아이콘으로 표시됨
- 효과 지속시간은 툴팁으로 확인 가능
- 보호막 효과는 우주선 주변에 푸른 원으로 시각화됨

## 향후 개선 사항

1. 더 다양한 아이템 추가 (공격 속도 증가, 점수 배수 증가 등)
2. 아이템 효과 시각화 개선 (화면 테두리 색상 변경 등)
3. 특별 이벤트 구현 (보스전, 아이템 샤워 등)
4. 아이템 효과 사운드 추가 및 개선
5. 파워업 아이템을 활용한 도전 과제 시스템 도입
