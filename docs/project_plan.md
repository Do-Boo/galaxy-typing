# Galaxy Typing 프로젝트 계획

## 📋 프로젝트 개요
- **프로젝트명**: Galaxy Typing
- **설명**: 우주 테마의 Flutter 타이핑 연습 앱
- **플랫폼**: 모바일 (iOS/Android), 웹 (Desktop/Tablet)

4. **몰입감 있는 경험**: 사운드, 애니메이션, 피드백의 조화

#### 2025.06.13 15:27:02 - 한글 조합 중 오타 평가 제외 개선
**개선 배경**: 
- 사용자 피드백: "조합 중엔 오타 평가를 안해야 맞는거 아냐?"
- 기존 방식의 문제: 조합 중인 문자에도 부분 점수를 주고 오타 카운트 진행

**개선 내용**:
1. **조합 중 문자 평가 제외**:
   - `_isInComposition()` 함수로 조합 상태 정확히 판별
   - 조합 중인 문자는 정답도 오타도 아닌 "평가 제외" 상태로 처리
   - 자모와 완성형 혼재, 초성/중성/종성 조합 과정 모두 감지

2. **정확도 계산 개선**:
   - 실제 평가된 문자 수(`evaluated`)만을 분모로 사용
   - 조합 중인 문자는 계산에서 완전히 제외
   - 더 정확하고 공정한 정확도 측정

3. **CPM 계산 단순화**:
   - 완성된 문자 + 평가 가능한 문자만으로 계산
   - 조합 제외로 더 안정적인 스무딩 (0.8~0.9 팩터)
   - 예측 기반 계산 제거로 더 정확한 측정

**결과**: 한글 조합 특성을 완벽 반영한 자연스럽고 정확한 타이핑 측정 시스템

## 🎯 기술적 세부사항

### 자소 단위 계산 예시
```
"컴퓨터" 입력 시:
- 컴: ㅋ + ㅓ + ㅁ = 3타
- 퓨: ㅍ + ㅠ = 2타  
- 터: ㅌ + ㅓ = 2타
- 총 7타

10초에 입력 시:
- 타수 = 7 × 60 / 10 = 42타/분
```

### 구현된 핵심 함수들
- `_getJasoCount()`: 텍스트의 총 자소 수 계산
- `_getKoreanJasoCount()`: 한글 문자의 자소 수 계산
- `_decomposeKorean()`: 한글 분해 (초성, 중성, 종성)
- `_compareTextByJaso()`: 자소 단위 텍스트 비교
- `_decomposeTextToJasos()`: 텍스트를 자소 리스트로 분해
