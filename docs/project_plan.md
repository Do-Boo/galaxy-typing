# Galaxy Typing 프로젝트 계획

## 📋 프로젝트 개요
- **프로젝트명**: Galaxy Typing
- **설명**: 우주 테마의 Flutter 타이핑 연습 앱
- **플랫폼**: 모바일 (iOS/Android), 웹 (Desktop/Tablet)
- **개발 언어**: Dart/Flutter

## 🎯 주요 목표
1. **생동감 있는 UI/UX**: 우주 테마의 인터랙티브한 사용자 경험
2. **플랫폼별 최적화**: 모바일과 웹 환경의 차이점을 고려한 구현
3. **정확한 타이핑 측정**: 한글 조합 과정을 고려한 정밀한 CPM/정확도 계산
4. **몰입감 있는 경험**: 사운드, 애니메이션, 피드백의 조화

#### 2025.06.13 15:19:50 - 한글 조합 CPM 계산 문제 해결
**문제점**: 
- 한글 조합 과정에서 미완성 상태가 오타로 인식되어 CPM이 부정확하게 계산됨
- "ㅇ" → "아" → "안" 과정에서 각 단계가 오류로 처리되어 정확도와 CPM이 지속적으로 하락

**해결 방안**:
1. **한글 문자 유사도 점수 시스템 구현**:
   - `_getKoreanMatchScore()` 함수로 초성(40%), 중성(40%), 종성(20%) 가중치 적용
   - 조합 중인 상태에 최소 80% 점수 보장
   - 자모와 완성형 혼재 시 90% 점수 부여

2. **정확도 계산 개선**:
   - 입력 길이 제한으로 과도한 입력 방지
   - 진행률 기반 보정으로 타이핑 진행도 반영 (최대 10% 보너스)
   - 정확도 100% 초과 방지

3. **CPM 계산 안정화**:
   - 진행률 기반 예상 완성 문자 수 계산
   - 적응적 스무딩으로 급격한 변화 완화 (변화량에 따라 0.7~0.85 팩터)
   - 비현실적인 값 제한 (최대 1000 CPM)

**결과**: 한글 타이핑 시 자연스러운 CPM 증가와 안정적인 정확도 유지
